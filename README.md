## Quem somos

Guilda de artesãos de software construindo ferramentas de IA para quem produz. Modelo de guilda com 6 pessoas — Mestre, Profissional, Aprendiz. Framework CL/CI/CD.

---

## O que construímos

### Sitia — Conselheira de gestao financeira rural

Gestao financeira conversacional para pequenos produtores rurais via WhatsApp. O produtor conversa naturalmente e recebe insights sobre receita, custo, margem e caixa.

- **Primeiro mercado:** Cariri, PB — caprinos, bovinos de leite, queijo artesanal
- **Planos:** Chacara R$49,90 / Sitio R$89,90 / Fazenda R$169,90
- **Meta 90 dias:** 50 produtores pagantes
- **Visao:** Gestao financeira -> contabilidade rural -> infraestrutura de decisao rural do Brasil

### Gepetto — Plataforma de agentes IA

Infraestrutura serverless para rodar agentes de IA em producao. 100% Cloudflare Workers.

Se e compartilhado entre verticais -> Gepetto. Se e especifico de dominio -> vertical.
